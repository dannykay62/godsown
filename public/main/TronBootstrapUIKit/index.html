Built with Less

Tron is built with Less and follow Bootstrap Less structure. This makes Tron more maintainable, and extendable. Easily customize colors, sizes and shapes with just few variables.

Absolutely Flat Design

With Tron your favorite Bootstrap elements becomes absolutely flat. No more gradients and shadows. Tron gives your website or app modern look and feel.

Mobile Friendly

By default Bootstrap provides you with powerful responsive grid system which helps you to build mobile first projects on the web. Tron extends this ability by adding tocuh-friendly components.

Extra Components

Besides default Bootstrap components which are totaly restyled Tron has to offer some extras. This will give you even more possibilities.

High Definition Ready

No more raster griphic (exept real images of course). Every single components is styled with pure CSS, all graphics are vector based. Tron will look sharp on any screen.

900+ Font Icons

Tron is shipped with 900+ font icons - FontAwesome and Flaticon Material Design Pack by Google. So when it comes to icons you have endless possibilities here.
